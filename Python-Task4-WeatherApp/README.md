# Weather Application

## Description
This Weather Application is developed using Python.

It allows users to enter a city name and view the current weather information.

## Features
- Search weather by city
- Temperature
- Humidity
- Pressure
- Wind Speed
- Weather Description
- Error Handling

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API
